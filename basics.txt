Setting Up and Basic commands

basic set-up before starting
git config --global user.name "username"
git config --global user.email "email"

basic commands
1.initialize a new git repo --> git init
2.create a new file --> touch filename


To Open File

1.Using vim command
vim filename
i --> to insert line
esc
to save and exit                     to save without changes
   type wq+enter                        q!+enter
Or :x+enter                          Or shift+ZQ
Or shift+ZZ

2.Using nano command
nano filename
ctrl+s --> to save
ctrl+x --> to exit


Creating and Managing branches
git branch branchname --> creates branch
git checkout branchname Or 
git switch branchname --> switch btw branches
git branch --> list all the branches
